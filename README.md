# Sliding Mode Control for BLDC Motor Speed Regulation  
**Bachelor's Thesis Project**  
*Politechnika Wrocławska, Wydział Elektryczny – Elektromechatronika*  
---
## Project Overview

This repository contains the complete development of a **robust speed controller** for a **Brushless DC (BLDC) motor** using **Sliding Mode Control (SMC)** techniques. 
The project compares classical SMC with the **Super-Twisting Sliding Mode Control (ST-SMC)** algorithm, which reduces chattering while maintaining robustness to disturbances and parameter uncertainties.

The implementation progresses in three main stages:
1. **Simulation** – MATLAB/Simulink models 
2. **Hardware Implementation** – STM32 Nucleo-F767ZI
3. **Future Enhancement** – RBF Neural Network-based adaptive control (AI integration)
